# Elixir Rsync

## Install

```bash
$ npm install elixir-rsync --save-dev
```

##Â Usage

### Example `gulpfile.js`

```js
var elixir = require( 'laravel-elixir' );

elixir( function( mix )
{
    mix.rsync( 'theme/src/', 'public/theme' );
});
```

### Advanced Example

```js
var elixir = require( 'laravel-elixir' );

elixir( function( mix )
{
    mix.rsync( 'theme/src/', 'public/theme', '-rc --delete' );
});
```
